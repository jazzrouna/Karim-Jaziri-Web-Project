1 - install node packages in both backend and frontend folders:
	1.1 : cd backend --> npm install
	1.2 : cd .. ; cd frontend --> npm install

2 - Install required dependencies in frontend :
	2-1 npm install react-redux
	2-2 npm install react-router-dom
	2-3 npm install axios
	2-4 npm install react-bootstrap bootstrap

3- Install required dependencies for backend :
	3-1 npm install react-joi-validation --save
	3-2 npm install cors
	3-3 npm install express
	3-4 npm install bcryptjs ( optional for password encryption ) 
	3-5 npm install mongodb
	3-6 npm install mongoose
4- Update your .env file & your API 

5- start app ( npm app.js "backend / npm start "frontend)
